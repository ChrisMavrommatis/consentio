üìù Integration Guide

Getting started with Consentio is simple:

1. Include Consentio

<script src="https://cdn.jsdelivr.net/npm/consentio/dist/consentio.min.js"></script>

2. Initialize

const consentio = new Consentio({ position: 'bottom', theme: 'light', onAccept: (consent) => { console.log('Consent accepted:', consent); // Enable tracking scripts }, onReject: (consent) => { console.log('Consent rejected:', consent); // Disable optional tracking } });

3. Google Tag Manager Integration

// Consentio automatically triggers GTM events // Listen for 'consentio_consent_updated' event in GTM // Access consent state via dataLayer
